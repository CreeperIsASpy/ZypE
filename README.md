### 世界观
1、世界：分为“粉色世界”与“黑色世界”。  
2、任务：原本生活于“粉色世界”的主角（某姓曹女子，即图标人物）遭遇车祸，坠入“黑色世界”后寻找“迷宫尽头”的彼岸。
